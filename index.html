Hilfe zu den drei Gleichungstypen

Kurz, klar und passend für Klasse 10.

1. Form: x² + q = 0

Stelle nach x² um: x² = -q. Ziehe dann die Wurzel. Es gilt also x = ±√(-q). Das klappt nur, wenn -q nicht negativ ist und eine passende Quadratwurzel hat.

2. Form: x² + p x = 0

Klammere x aus: x(x + p) = 0. Nutze dann die Nullproduktregel: Entweder x = 0 oder x + p = 0, also x = -p.

3. Form: x² + p x + q = 0

Suche zwei Zahlen, deren Summe p und deren Produkt q ist. Mit diesen Zahlen kann man die Gleichung in Faktoren zerlegen, zum Beispiel (x + a)(x + b) = 0.

Eingabe-Tipp

Eine Lösung kannst du einfach als Zahl eingeben. Zwei Lösungen trennst du mit Semikolon, Komma oder Leerzeichen, zum Beispiel 2; -3. Die Reihenfolge ist egal.
